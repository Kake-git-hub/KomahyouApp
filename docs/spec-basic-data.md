# 確定仕様：基本データ画面（To-Be 正本）

> 全体見直しセッション（2026-06-08）でオーナー確認済みの「目標仕様」。
> ③（通常授業＝テンプレ方式）と整合。現行から項目を削ぎ落とす方向。

## A. タブ構成

- タブは **生徒／講師／教室データ** の3つ。
  - ★**マネージャータブは削除**（使用していない）。
- 各タブの表は**並び替え・フィルタ**ができる。
- ペア制約は自動割振ルール画面で管理（⑧）。

## B. 生徒タブ

- 入力項目：**名前・表示名・メール・入塾日・退塾日・生年月日**。
  - ★**「非表示（isHidden）」項目は廃止**（手動の非表示フラグは持たない）。
- 学年は**生年月日から自動判定**（学年欄は持たない）。
- 在籍／非在籍は**自動判定**：
  - **入塾日より前・退塾日より後・高3卒業後**は「**非在籍**」として扱う。
  - ★**高3卒業後は自動的に非在籍**（卒業判定を生年月日から行う）。
- 非在籍の生徒は盤面・各種集計から外れる。

## C. 講師タブ

- 入力項目：**氏名・表示名・メール・入塾日・退塾日・担当科目（科目＋担当上限学年）**。
  - ★**「出勤可能コマ」「メモ」「非表示」は廃止**（出勤枠はテンプレで管理するため不要）。
- 担当科目は「**選んだ学年以下を担当可能**」（上限学年方式）。
- **在籍講師／非在籍講師の表示切替**がある（非在籍は入塾日・退塾日から自動判定）。

## D. 教室データタブ

- 設定するのは **定休日（休校曜日）と机数**。
- 役割分担：
  - **初回は「初期設定フロー」（②）で設定**。
  - **途中変更はこの「教室データタブ」で行う**（両方残す）。

## E. 通常授業（③の整合）

- 基本データに**通常授業の編集タブは無い**。通常授業は**コマ表のテンプレモード**で編集。
- Excelには「**通常授業テンプレ**」シートがあり、テンプレを書き出し・取り込みできる。

## F. Excel取込・出力（実体は「バックアップ/復元/初期設定」画面）

- 管理データ（生徒・講師・教室データ・通常授業テンプレ）を**テンプレ出力／現データ出力／初期取込／差分取込**できる。
- 差分取込は**ID列を残して再取込すると、講習・ルール・盤面・ストックを保持したまま更新**。
- ★Excel運用は**初回セットアップで必要なので残す**が、③のテンプレ方式に合わせて**シート構成を整理**する（旧 RegularLessonRow 由来の不要部分を除去）。

## G. 祝日・強制開校日

- ★**holidayDates（個別休日）／forceOpenDates（強制開校日）は「盤面」で設定する**方針。
- **基本データ画面には祝日・強制開校日の編集UIは一切置かない。**
